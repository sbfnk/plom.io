Getting started

We assume that you have added simforence-population-based to you PYTHONPATH

simforence-population-based contains:

  • python code to generate model in plain C. This code lives in simforence_model_builder/
  • C code to run models. This code lives in templates/src/C/ The C code contain generic part and model specific part that are rendered using model_builder

Generating documentation for the C code:

in doc/ run: doxygen Doxyfile and open doc/html/index.html with a web browser.

Using the models and inference methods

Please refer to Simforence documentation (https://github.com/simforence/simforence-web)
